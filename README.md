# 🎨 高级颜色选择器

基于 **PyQt6** 开发的桌面端颜色选择工具，支持 **RGB**、**HEX**、**HSV** 三种色彩模式，所有模式下均可通过滑块、输入框或鼠标滚轮实时调整颜色，并提供实时预览与一键复制功能。

## ✨ 功能特性

- **三种色彩模式**
  - RGB：红 / 绿 / 蓝（0–255），带滑块
  - HEX：十六进制颜色码（如 `#FF5733`），支持复制到剪贴板
  - HSV：色相（0–359）/ 饱和度（0–255）/ 明度（0–255），带滑块

- **实时同步** — 任意模式下修改颜色，所有其他模式和预览区同步更新
- **鼠标滚轮支持** — 在输入框上滚动鼠标滚轮即可微调数值
- **三种修改颜色方式** — 直接输入、鼠标滚轮滑动、滑块拖拽
- **一键复制** — HEX 页面提供复制按钮，点击后将十六进制颜色码复制到剪贴板，并提供反馈提示
- **颜色预览** — 底部大色块实时展示当前选中的颜色
- **简洁直观的 UI** — 标签页式布局，固定窗口大小（500×700）

## 🚀 使用说明

| 操作 | 说明 |
|------|------|
| **输入框调色** | 直接在数值输入框中键入数字，其他输入框和颜色预览会自动更新 |
| **滑块调色** | 拖动滑条快速调整各通道数值 |
| **滚轮微调** | 鼠标悬停在输入框上，滚动滚轮以 ±1 步长微调 |
| **切换模式** | 点击顶部标签页（RGB / HEX / HSV）切换不同色彩空间 |
| **复制颜色** | 在 HEX 页输入或调好颜色后，点击「📋 复制十六进制到剪贴板」按钮 |
| **预览颜色** | 底部大色块始终显示当前颜色的实际效果 |


## 🛠 技术细节

- 使用 **PyQt6** 构建 GUI 界面
- 自定义 `NumberLineEdit` 控件：限制 0–255 数值输入，支持鼠标滚轮微调
- 自定义 `HexLineEdit` 控件：限制 7 位 (#RRGGBB) 输入，支持鼠标滚轮微调
- 信号/槽机制实现各组件之间的双向同步
- `_updating` 标志位防止循环更新的递归问题
- `QColor.fromHsv()` / `QColor.getHsv()` 实现 HSV ↔ RGB 转换
